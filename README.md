## 🚀 Getting Started

### Prerequisites
- A C compiler (GCC recommended)

```bash
# Check if GCC is installed
gcc --version
```

### Compile & Run Any Program

```bash
# Clone the repository
git clone https://github.com/Sagarchaudhary91187/C-Programming.git
cd C-Programming

# Compile any .c file
gcc filename.c -o output

# Run
./output
```

### Example

```bash
gcc 1_Basics/1_Basic_code.c -o hello
./hello
```
